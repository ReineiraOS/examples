# Decisions — 2026-04-22

## Stack & Scope (locked)

- **Chain:** Arbitrum (mainnet Q3 2026). Testnet: Arbitrum Sepolia (Fhenix testnet). Rationale: FHE support + L2 cost profile + existing ReineiraOS deployment.
- **Settlement token:** Stablecoin-agnostic via `IFHERC20`. USDC primary. EURC for EU issuers. Rationale: issuer choice, MiCA fit, multi-venue via CCTP v2.
- **Wallet (human counterparties):** ZeroDev ERC-4337 smart accounts + passkey auth. Rationale: no seed-phrase UX for non-crypto-native issuers + MM traders.
- **Wallet (treasuries):** Safe multisig. Rationale: standard for token foundations, already held by most issuers.
- **Upgradeability:** UUPS for custom plugins. Dual-mode public→encrypted upgrade path preserved. Rationale: protocol standard; enables hot-patching without re-bonding existing engagements.

- **Advisory:** NGFA pays ReineiraOS for architecture review + plugin authoring + test harness + mainnet migration. Scoped, not equity.
